Steps to start the applications

Part 1:-Angular

1. git clone the source code from github
2. Do npm install for the application
3. In the terminal, run the npm start or ng serve -o command
4. Access the application in localhost:4200
5. Check the login component for reference.

Part 2:- Node JS

1. git clone the source code from github
2. Do npm install.
3. In the terminal, run the node index.js command
4. The mongo DB is accessed from localhost:27017
